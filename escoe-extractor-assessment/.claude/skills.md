# Skill: ESCoE Taxonomy Extraction
- **Expertise**: Mapping job description text to the ESCO (European Skills, Competences, Qualifications and Occupations) framework.
- **Task**: Extract "Knowledge", "Skills", and "Attitudes" using the ESCoE vector-based approach.
- **Goal**: Provide standardized, research-grade skill labels for IT job descriptions.
- **Constraint**: Use the [ESCoE Skills Extractor Library](https://github.com) patterns for semantic matching.
